#DOCKER COMMANDS

## - Build Images
docker build [OPTIONS] PATH

## - Run Images
docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

## - Get Logs
docker logs CONTAINER_ID

## - List Images
docker images

## - List Containers
docker ps

## - Tag Images
docker tag SOURCE_IMAGE[:TAG] TARGET_IMAGE[:TAG]

## - Login to DockerHub
docker login

## - Push Images
docker push NAME[:TAG]

## - Pull Images
docker pull NAME[:TAG]
